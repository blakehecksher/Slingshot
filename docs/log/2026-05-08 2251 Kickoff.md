## Decisions made

- TypeScript over JS (math safety on physics code).
- `@dimforge/rapier3d-compat` over raw `rapier3d` (simpler WASM init, perf cost negligible for Phase 1).
- Vite as bundler.
- GitHub Pages via official Actions (`upload-pages-artifact` + `deploy-pages`), not third-party `gh-pages`.
- Custom gravity computation; Rapier global gravity stays at zero. Phase 1 has no collisions but uses Rapier rigid bodies so Phase 2 doesn't require integrator rewrite.
- Fixed-timestep physics (1/120s) with render interpolation.
- Camera shake on camera node only — never on ship body.
- Gravity formula uses softening: `F = G * m1 * m2 / (r^2 + ε^2)`.
